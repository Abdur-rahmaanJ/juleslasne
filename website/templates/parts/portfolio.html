This very website

Yes, that's right!
This very website is one of the project I'm the most proud of.
I've purchased the domain name juleslasne along with the TLDs .com .net .fr and .me for it.
I've launched an EC2 instance on AWS, configured the DNS', Route 53, SSL for HTTPs and this website was off to the races !

The backend for this website runs with Flask.
The frontend is currently handled by HTML and CSS with Bootstrap.
You can checkout it's source code here.

21-42sh

21-42sh is a 42 project made in collaboration with gpouyat, mleroy and chbravo-.
For this project, we had to create from scratch a CLI with some POSIX requirements.
For more info, check out it's Github Repository.

IntraPy

IntraPy is a package made in collaboration with @cvermand. The goal of IntraPy was to simplify the usage of 42's API by making a python package for it. You can check out it's source code here.

ShellGrammarGenerator

This script, made in collaboration with gpouyat,
converts a yacc file into a 3D array and a header file for a modular C parser. For more info, you can check out it's source code here.
